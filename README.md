# 简介

Spring是构建企业级应用的一个轻量级框架。Spring由多个模块组成，但我们可以轻易的挑选应用所需的模块，而不用引用不需要的东西。Spring提供了IOC容器，声明式事务处理，通过RMI或者Web Service远程访问，并支持多种持久化技术，如Hibernate，JPA，Mybatis等。还提供了功能齐全的Web层MVC框架，以及AOP等。
